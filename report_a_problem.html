Report a Problem

Wait! Before you file a bug report, here are some known issues that are currently being fixed. Please do not file a new bug report for these problems:

  • Opening the menu on iOS/Android and then rotating the device might cause a blank spot at the top of the page.

  • Footer might be visible on pages with small amounts of text.


If your issue is not listed above, click the button below to submit a new bug report.


Report


Please describe the bug in as much detail as you can and list any step(s) necessary to reproduce the issue.

Also, be sure to list the following information:

  • Site version (found here) under "Site Information".

  • The browser you are using, such as:

    • Google Chrome
    • Mozilla Firefox
    • Apple Safari
    • Opera
    • Microsoft Edge
    • Other (please specify)

  • The operating system you are using, such as:

    • Android
    • Windows
    • iOS/iPadOS
    • macOS
    • ChromeOS
    • Linux (be sure to list which distro you're using also)
    • Other (please specify)

The more information you provide, the better I am able to diagnose the problem.
